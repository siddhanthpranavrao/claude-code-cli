# Sample Project

This is a synthetic Python codebase for testing semantic and lexical search.

It intentionally contains:

- deep function call chains
- classes with multiple methods
- repeated domain keywords for BM25 search
- explicit error names and exception messages
- larger orchestration files
- small utility modules
- cross-package imports

Useful search terms:

- `EmbeddingDimensionMismatch`
- `stale_vector_index`
- `tenant isolation`
- `chunk overlap`
- `citation drift`
- `RateLimitExceeded`
- `hybrid score fusion`
- `dead letter queue`
- `PII redaction`
- `query rewrite`
- `semantic reranking`
